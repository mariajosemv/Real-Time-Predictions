Real time prediction

This is a linear regression based on a neural network model.
For this demo I am using one hidden layer for predict the cost of a
house given a number of rooms.

How does it works:

When you click on Train the model, the training loop will start
and on the right side you will see how the learning process is evolving,
if you feel satisfied with the loss and mse values, then Stop the training,
go to the Visor tab and Show the predictions!







Did you like the results?
Then Save the model so you can use it later.



Or if you want, you can also evaluate your own model!
